# Chess Game

A simple web-based chess game implemented using HTML, CSS, and JavaScript.

## Features

- Interactive chess board with 8x8 grid
- Chess pieces with proper positioning
- Basic game rendering and event handling

## How to Run

1. Clone or download the project.
2. Open `index.html` in your web browser.
3. Enjoy playing chess!

## Project Structure

- `index.html`: Main HTML file
- `index.js`: Entry point JavaScript file
- `Data/`: Contains game data and logic
  - `data.js`: Board initialization
  - `data.json`: Game data (if any)
  - `pieces.js`: Chess piece definitions
- `Events/`: Event handling
  - `global.js`: Global event listeners
- `Helper/`: Utility functions
  - `constants.js`: Constants
- `images/`: Image assets
  - `pieces/`: Chess piece images (black and white)
- `Render/`: Rendering logic
  - `main.js`: Main rendering functions
- `styles/`: CSS styles
  - `index.css`: Main stylesheet

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Modules)
